Common Text Modifications

Type Description
Bold Text Use bold sparingly for emphasis. If you’re using a sans serif font, skip italic and use bold for emphasis. With a serif font, use italic for gentle emphasis, or bold for heavier emphasis.
Italic Text Use Italics when you want to emphasize a certain word or phrase. A common use for italics is to draw attention to a particular part of a text in order to provide emphasis. If something is important or shocking, you might want to italicize that word or phrase so that your readers don't miss it.With a serif font, use italic for gentle emphasis, or bold for heavier emphasis.
Underlined Text Default look for links.
Spaced Text: Useful, because it may help people with difficulties read better.
